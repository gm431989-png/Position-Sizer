How it works
You log into Angel One once a day and paste your JWT token here. The app uses it to fetch live prices for your alerts. Token expires daily — re-paste each morning before trading.
CORS limitation: Direct API calls from browser are blocked by Angel One's servers. Use Option B (Cloudflare Worker) for automatic login, or paste your JWT token manually below.
How to get your JWT token
1
Go to smartapi.angelbroking.com → login → create an app → copy your API key
2
Enable TOTP in Angel One app: Profile → Security → Enable TOTP → scan QR with Google Authenticator
3
Each morning, run this in browser console on any page:
fetch('https://apiconnect.angelbroking.com/rest/auth/angelbroking/user/v1/loginByPassword',{method:'POST',headers:{'Content-Type':'application/json','X-UserType':'USER','X-SourceID':'WEB','X-ClientLocalIP':'CLIENT_LOCAL_IP','X-ClientPublicIP':'CLIENT_PUBLIC_IP','X-MACAddress':'MAC_ADDRESS','X-PrivateKey':'YOUR_API_KEY'},body:JSON.stringify({clientcode:'YOUR_CLIENT_ID',password:'YOUR_MPIN',totp:'YOUR_TOTP'})}).then(r=>r.json()).then(d=>console.log('JWT:',d.data?.jwtToken))
4
Copy the JWT token from console → paste above → click Connect
What this does
A tiny free Cloudflare Worker acts as a proxy — it handles the Angel One login for you, bypassing the CORS restriction. One-time 5-minute setup, then fully automatic. Your credentials stay only in your Cloudflare account.
Cloudflare Worker code 
export default {
  async fetch(request, env) {
    const corsHeaders = {
      'Access-Control-Allow-Origin': '*',
      'Access-Control-Allow-Methods': 'POST, GET, OPTIONS',
      'Access-Control-Allow-Headers': 'Content-Type',
    };
    if (request.method === 'OPTIONS')
      return new Response(null, { headers: corsHeaders });

    const { clientcode, password, totp, apikey } = await request.json();

    const res = await fetch(
      'https://apiconnect.angelbroking.com/rest/auth/angelbroking/user/v1/loginByPassword',
      {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'X-UserType': 'USER',
          'X-SourceID': 'WEB',
          'X-ClientLocalIP': '127.0.0.1',
          'X-ClientPublicIP': '127.0.0.1',
          'X-MACAddress': '00:00:00:00:00:00',
          'X-PrivateKey': apikey,
          'Accept': 'application/json',
        },
        body: JSON.stringify({ clientcode, password, totp }),
      }
    );
    const data = await res.json();
    return new Response(JSON.stringify(data), {
      headers: { ...corsHeaders, 'Content-Type': 'application/json' },
    });
  },
};
